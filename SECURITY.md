# Security

If you discover a security vulnerability in tenter-rs, please report it through one of the channels below.

## Reporting

For sensitive issues (anything that could expose users if disclosed publicly), use [GitHub's private vulnerability reporting](https://github.com/goweft/tenter-rs/security/advisories/new) to disclose confidentially.

For non-sensitive issues, open a GitHub issue with the `security` label.

## Disclosure

I aim to acknowledge reports within a few days and to keep you updated through remediation. Please allow a reasonable window to address the issue before public disclosure.
